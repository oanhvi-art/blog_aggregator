# gator

A production-ready, open-source RSS feed aggregator CLI built with Node.js, TypeScript, Drizzle ORM, and PostgreSQL.

## Motivation
gator was built to provide a lightweight, terminal-based solution for aggregating and consuming RSS feeds efficiently. By leveraging TypeScript and a relational database like PostgreSQL, it ensures robust data handling, seamless feed switching, and rapid command execution without the overhead of heavy web-based UIs.

## Quick Start
Get a local instance up and running in less than 2 minutes.

### Prerequisites
Before running this program, ensure you have the following installed on your system:
* Node.js (v18 or higher)
* PostgreSQL server running locally
